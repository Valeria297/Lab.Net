# Лабораторные работы по C#

В данном репозитории представлены 5 лабораторных работ, выполненных в ходе изучения основ объектно-ориентированного программирования на C#.  
Каждая работа находится в отдельной папке и представляет собой готовое консольное приложение.

## Структура репозитория

📁 **Lab1** — основы ООП: классы, объекты, методы. Реализован простой калькулятор.  
📁 **Lab2** — наследование, коллекции (`List<T>`), обработка исключений.  
📁 **Lab3** — массивы (одномерные, прямоугольные, зубчатые), перечисления (`enum`), события, свойства.  
📁 **Lab4** — перегрузка методов и операторов, конструкторы (в том числе статический и копирования), деструкторы.  
📁 **Lab5** — работа с файлами: чтение параметров оборудования из текстового файла, вывод в консоль, сохранение отчёта.  
📁 **Lab6** — Windows Forms приложение: таблица параметров оборудования в памяти, вывод в список, добавление новых параметров.

## Как запустить

 Перейдите в папку нужной лабораторной и выполните:  
 cd Lab1  
 dotnet run  
 Либо укажите проект из корня:  
 dotnet run --project Lab1  

Все проекты написаны на C# и используют .NET.  
Для запуска необходимо установить .NET SDK.
